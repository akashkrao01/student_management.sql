Student Management System

SQL Database Project

Project Description

This project demonstrates a SQL-based student management database that stores student details and scores and performs queries such as average score calculation, top performer identification, grouping students by grade, and updating records.

Features

  • Create Student Database
  • Store Student Records
  • Calculate Average Scores
  • Identify Top Performer
  • Group Students by Grade
  • Update Student Records

Project Files

  • student_management.sql – SQL database script
